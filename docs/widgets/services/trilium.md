---
title: Trilium
description: Trilium Widget Configuration
---

Learn more about [Trilium](https://github.com/TriliumNext/Notes).

Find (or create) your ETAPI key under `Options > ETAPI > Create new ETAPI token`.

```yaml
widget:
  type: trilium
  url: https://trilium.host.or.ip
  key: etapi_token
```
