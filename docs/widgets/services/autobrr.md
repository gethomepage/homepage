Find your API key under `Settings > API Keys`.

Allowed fields: `["approvedPushes", "rejectedPushes", "filters", "indexers"]`.

```yaml
widget:
  type: autobrr
  url: http://autobrr.host.or.ip
  key: apikeyapikeyapikeyapikeyapikey
```
