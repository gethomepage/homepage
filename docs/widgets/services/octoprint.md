Learn more about [OctoPrint](https://octoprint.org/).

Allowed fields: `["printer_state", "temp_tool", "temp_bed", "job_completion"]`.

```yaml
widget:
  type: octoprint
  url: http://octoprint.host.or.ip:port
  key: youroctoprintapikey
```
