---
title: Grafana
description: Grafana Widget Configuration
---

Learn more about [Grafana](https://github.com/grafana/grafana).

Allowed fields: `["dashboards", "datasources", "totalalerts", "alertstriggered"]`.

```yaml
widget:
  type: grafana
  url: http://grafana.host.or.ip:port
  username: username
  password: password
```
