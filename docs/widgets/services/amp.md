---
title: AMP
description: AMP Widget Configuration
---

Learn more about [AMP](https://cubecoders.com/AMP).

The username and password are the same as used to login to the web interface.

Allowed fields: `["state", "users", "cpu", "memoryUsedPercent", "memoryUsed"]`.

```yaml
widget:
  type: amp
  url: http://amp.host.or.ip
  username: admin
  password: password
```
