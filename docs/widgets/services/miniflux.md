---
title: Miniflux
description: Miniflux Widget Configuration
---

Learn more about [Miniflux](https://github.com/miniflux/v2).

Api key is found under Settings > API keys

Allowed fields: `["unread", "read"]`.

```yaml
widget:
  type: miniflux
  url: http://miniflux.host.or.ip:port
  key: minifluxapikey
```
